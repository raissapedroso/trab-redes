# Análise Comparativa entre HTTPS e HTTP/3

## Disciplina
Redes de Computadores
## Aluna
Raissa Pedroso
## Proposta do Trabalho
Este trabalho propõe uma análise comparativa entre o HTTPS tradicional e o HTTP/3, com foco no desempenho, segurança e comportamento das conexões.

Pretende-se analisar o funcionamento do HTTPS baseado em TLS sobre TCP e compará-lo com o HTTP/3, que utiliza o protocolo QUIC com TLS 1.3 integrado. A análise buscará identificar diferenças no tempo de conexão, latência e desempenho da comunicação.

Ferramentas de monitoramento de tráfego, como o Wireshark, serão utilizadas para auxiliar na análise prática das conexões.
## Tecnologias Pretendidas
* HTML / CSS
* JavaScript
* Node.js
* Git/GitHub
## Estrutura Inicial
O projeto pretende conter:
*Implementação de conexões HTTPS e HTTP/3
*Comparação entre modelos de comunicação (TCP + TLS vs QUIC + TLS integrado)
*Testes de desempenho (latência e tempo de resposta)
*Observação prática do tráfego de rede
