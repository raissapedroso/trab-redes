# Análise Comparativa entre HTTP e HTTPS

## Disciplina
Redes de Computadores
## Aluna
Raissa Pedroso
## Proposta do Trabalho
Este trabalho propõe uma análise comparativa entre os protocolos HTTP e HTTPS, com foco em segurança e desempenho das conexões. Pretende-se também observar diferenças entre TLS 1.2 e TLS 1.3, utilizando ---- e ferramentas de monitoramento de tráfego para auxiliar na análise prática das conexões.
## Tecnologias Pretendidas
* HTML / CSS
* JavaScript
* Node.js
* Git/GitHub
## Estrutura Inicial
O projeto pretende conter:
* Implementação de conexões HTTP e HTTPS
* Comparação entre TLS 1.2 e TLS 1.3
* Testes de desempenho
* Observação prática do tráfego de rede
